
- 错误信息：
```bash
...
Error: connect ETIMEDOUT xxx.xxx.xx.xxx:443
...
```

- 解决方案：
先 `ping` 下那个IP，如果可以`ping`通就添加下面的参数
- 例子：
```
yarn add -D webpack --network-concurrency 1
或者
yarn --network-concurrency 1
```
